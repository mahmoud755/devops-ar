عن المنصة

مسار واضح

متابعة

جلسات أسئلة وأجوبة + توجيه في المشاريع + مراجعة شغل المتدرب.

محتوى التدريب

هنا بتحط Outline مختصر. التفاصيل الكتير ممكن تكون في PDF أو بوستات على الفيس.

Module 1: Linux + Networking Basics

Permissions, systemd, logs, ssh, basic networking, troubleshooting.

Module 2: Docker + Containers

Images/containers, Dockerfile, compose, best practices, registry basics.

Module 3: Kubernetes

Pods, deployments, services, ingress basics, configmaps/secrets, helm intro.

Module 4: CI/CD + Cloud (AWS)

Pipelines, GitHub Actions/Jenkins (اختيارك), IAM basics, EC2/VPC overview, deploy flow.

الأسئلة الشائعة

جاوب عن أهم اعتراضات الناس: أبدأ منين؟ محتاج خبرة؟ هشتغل امتى؟

هل لازم أكون Developer عشان أبدأ DevOps؟

مش شرط، لكن لازم أساسيات Linux + Networking، وبرمجة بسيطة مفيدة جدًا.

هتحتاج كام ساعة أسبوعيًا؟

أفضل نتيجة مع 6–10 ساعات أسبوعيًا (تعلم + تطبيق).

هل التدريب عملي؟

آه: Labs + مشروع نهائي. الهدف Portfolio تقدر تعرضه.

هل في متابعة؟

في Q&A وجروب دعم، وبحسب الباقة في متابعة على المشروع.
